#### SQLite Precompiled Binary for Android armeabi-v7a (ARM aarch64) 

```
$ wget https://raw.githubusercontent.com/rojenzaman/sqlite3-arm-aarch64/main/sqlite3
$ chmod 750 sqlite3
$ ./sqlite3 --version
```
